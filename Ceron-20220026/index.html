Switch

Un switch o conmutador es un dispositivo de interconexión utilizado para conectar equipos en red formando lo que se conoce como una red de área local (LAN) y cuyas especificaciones técnicas siguen el estándar conocido como Ethernet (o técnicamente IEEE 802.3).

Router

Un router es un dispositivo que conecta dos o más redes o subredes de conmutación de paquetes.

Tarjeta de red

Una tarjeta de red es un dispositivo que se instala en el interior de un ordenador para que este pueda conectarse a Internet. Según el tipo de tarjeta, esta puede incluir en su parte posterior un puerto para introducir el cable de Ethernet o admitir la conexión inalámbrica.

Interruptor inteligenete Wifi

Dispositivo inteligente que permite el control ON – OFF (conexión y desconexión) de la corriente eléctrica en toda clase de circuitos eléctricos con cargas resistivas e inductivas.

Access Poin

éestablecer una conexión inalámbrica entre equipos y pueden formar una red inalámbrica externa (local o internet) para interconectar dispositivos móviles o tarjetas de red inalámbricas.

5. Herramienta utilizada para realizar un cableado estructurado para, agregar imagen y descripción téecnica de cada una de ellas

Ponchadora

La ponchadora es una herramienta manual utilizada para insertar y asegurar los cables en los conectores de red (como los RJ-45). Es fundamental para crear conexiones de red confiables.

Crimpadora

La crimpadora es una herramienta similar a la ponchadora, pero se utiliza para unir un conector con un cable, específicamente aplastando o deformando el conector para asegurar su conexión con el cable. Es común en la terminación de cables coaxiales o de par trenzado.

Desforradora

La desforradora es una herramienta utilizada para remover el recubrimiento externo de los cables sin dañar los conductores internos. Es esencial para preparar los cables antes de conectarlos o crimpados.

Tester/Probador de cable

El tester o probador de cables es un dispositivo utilizado para verificar la continuidad y la correcta conexión de los cables de red. Permite identificar fallos en la transmisión de datos antes de que se instalen o utilicen en una red.

6. ¿Qué es el cableado horizontal?

El cableado horizontal es la parte del sistema de cableado estructurado que conecta las estaciones de trabajo al cuarto de telecomunicaciones o al panel de conexión. Este tipo de cableado generalmente se extiende horizontalmente a través de un piso o área específica del edificio.

7. ¿Qué es el cableado vertical o backbone?

El cableado vertical, también conocido como backbone, es el sistema de cableado que conecta los cuartos de telecomunicaciones entre diferentes pisos de un edificio o entre diferentes edificios en un campus. Este cableado suele transportar grandes cantidades de datos y es crucial para la interconexión de toda la red.

8. Tipos de cables para el sistema de cableado horizontal

Cable UTP (Unshielded Twisted Pair)

El cable UTP es un tipo de cable de par trenzado sin apantallar. Es comúnmente utilizado en redes de área local (LAN) debido a su bajo costo y facilidad de instalación.

Cable FTP (Foiled Twisted Pair)

El cable FTP es similar al UTP, pero incluye una capa de protección adicional contra interferencias electromagnéticas, gracias a un blindaje de aluminio que envuelve los pares de cables

Cable de Fibra Óptica

: La fibra óptica es un cable que transmite datos mediante pulsos de luz, lo que le permite alcanzar mayores velocidades y distancias en comparación con los cables de cobre tradicionales. Es ideal para redes que requieren alta velocidad y larga distancia.

9. Normativa para hacer cables de red (clase A y B)

Las normas T568A y T568B son estándares para el cableado de redes de par trenzado. Determinan el orden de los colores de los cables en el conector RJ-45, asegurando la compatibilidad y el rendimiento de la red. La principal diferencia entre ambas es el orden de los pares de colores.

10. Para qué se utiliza un cable directo en la red

Un cable directo, o cable patch, se utiliza para conectar diferentes dispositivos de red, como una computadora a un switch, router o hub. Es el tipo de cable más común en las conexiones LAN.

11. Para qué se utiliza un cable cruzado en la red

Un cable cruzado se utiliza para conectar dispositivos de red similares entre sí, como dos computadoras o dos switches directamente. En un cable cruzado, los pares de transmisión y recepción están invertidos.

  • ¿Qué son y para qué sirven las normas internacionales?

    • Las normas internacionales son directrices y estándares que buscan asegurar la calidad, seguridad y eficiencia en productos y servicios a nivel global. Un ejemplo de una normativa aplicada a redes informáticas es la norma ISO/IEC 11801, que especifica los requisitos para el cableado de telecomunicaciones en edificios.
